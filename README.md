## Requirements :

- nodeJS


## Quickstart :

1. `npm install`
1. `grunt`

Then copy "build/game.min.js" to "app/js/".

If you'd like to continuously update the game while hacking on the code, run the `watch` Grunt task:

```bash
grunt watch
```
